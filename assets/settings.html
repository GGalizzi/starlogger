Setting the path to your universe folder will allow the application to automatically add new planets you visit to the list.

Default locations (click one to automatically add it):

Windows:
C:\Program Files\Steam\steamapps\common\starbound\universe

MacOS:
~/Library/Application Support/Steam/SteamApps/common/Starbound/universe

32-bit Linux:
~/.local/share/Steam/SteamApps/common/Starbound/linux32/universe

64-bit Linux:
~/.local/share/Steam/SteamApps/common/Starbound/linux64/universe
